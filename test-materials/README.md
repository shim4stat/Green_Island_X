# テスト用素材

このフォルダには、証拠ベースの CO₂ 削減量記録機能をテストするためのサンプル画像が含まれています。

## 📁 ファイル一覧

- **Sample1.png** - 電気料金明細のサンプル
- **sample2.jpg** - 歩数記録のサンプル
- **sample3.jpg** - 太陽光発電記録のサンプル

## 🧪 使用方法

1. DAO 詳細ページの「証拠付きで削減量を記録する」セクションにアクセス
2. これらのサンプル画像をアップロード
3. 必要に応じて個人情報をマスキング
4. 証拠タイプを選択して削減量を計算
5. ブロックチェーンに記録

## ⚠️ 注意

これらの画像はテスト用のサンプルです。実際の証拠画像をアップロードする際は、個人情報を適切にマスキングしてください。
